# Muhammad Shakeeb Khalid | Portfolio

A personal developer portfolio for Muhammad Shakeeb Khalid, a Full Stack MERN Developer. It showcases professional experience, skills, education, and selected full-stack projects.

## Highlights

- Responsive React portfolio built with Vite and Tailwind CSS
- Animated interface powered by Framer Motion
- Interactive 3D MERN-stack visual using Three.js and React Three Fiber
- Project gallery with screenshots and a built-in resume viewer
- Ready for static deployment on Netlify

## Tech Stack

- React 19 and Vite
- Tailwind CSS 4
- Framer Motion
- Three.js, React Three Fiber, and Drei
- PDF.js

## Featured Projects

- **VendorHub** — Vendor management and quotation comparison platform.
- **Communo** — Community services and marketplace platform with role-based access and real-time messaging.
- **Multi-Vendor Service Marketplace** — Freelance-service marketplace with a multi-stage project workflow.

## Run Locally

The frontend application is in the `portfolio` folder.

```bash
cd portfolio
npm install
npm run dev
```

Open the local URL shown by Vite, usually `http://localhost:5173`.

## Available Commands

Run these from the `portfolio` folder:

```bash
npm run dev      # Start the development server
npm run build    # Create a production build in dist/
npm run preview  # Preview the production build
npm run lint     # Run Oxlint
```

## Deployment

This project includes a Netlify configuration in the repository-root `netlify.toml`.

- Base directory: `portfolio`
- Build command: `npm run build`
- Publish directory: `dist`

## Customization

Portfolio text, skills, experience, and projects are maintained in:

```text
portfolio/src/data/content.js
```

Project screenshots are stored in `portfolio/public/images/`.
